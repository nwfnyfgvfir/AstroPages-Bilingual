1，打开[New API](https://napi.azxue.de/)，点击右上角注册。输入你的用户名，邮箱，密码，完成人机验证后点获取验证码，输入验证码点击注册。

2，用第一步注册的邮箱或用户名登入账号，将你的用户名发给闲鱼商家，让他给你分配订阅（比如日卡，周卡，月卡）。

3，在钱包管理处确认订阅已激活。在左侧找到令牌管理，点添加令牌。\
